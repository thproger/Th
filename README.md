# 📋 Task Manager Telegram Bot

Телеграм-бот для управління завданнями з ролевою системою та MongoDB.

## 🗂 Структура проекту

```
telegram_bot/
├── main.py              # Точка входу
├── config.py            # Конфігурація, ролі, статуси
├── database.py          # MongoDB (motor) — всі запити
├── keyboards.py         # Reply та Inline клавіатури
├── utils.py             # Форматування виводу
├── handlers/
│   ├── __init__.py      # Реєстрація всіх роутерів
│   ├── registration.py  # /start, реєстрація, призначення ролей
│   ├── tasks.py         # Створення/перегляд/оновлення завдань
│   └── groups.py        # Управління групами
├── requirements.txt
└── .env.example
```

## ⚙️ Налаштування

### 1. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 2. Змінні середовища
Скопіюй `.env.example` у `.env` та заповни:
```
BOT_TOKEN=your_telegram_bot_token_here
MONGO_URI=mongodb://localhost:27017
DB_NAME=task_manager_bot
```

### 3. MongoDB
Переконайся що MongoDB запущена. Локально:
```bash
mongod --dbpath /data/db
```
Або використай MongoDB Atlas (хмарна).

### 4. Запуск
```bash
python main.py
```

## 👑 Ролева система

| Роль | Права |
|------|-------|
| **Адмін** | Все: призначення ролей, перегляд усіх завдань/груп, будь-яке управління |
| **Керівник** | Видавати завдання будь-якому зареєстрованому користувачу, переглядати свої видані завдання |
| **Керівник групи** | Формувати свою групу (з рядових), видавати завдання учасникам, переглядати завдання групи |
| **Рядовий** | Переглядати свої завдання, змінювати їх статус, залишати коментарі |

## 📋 Функціонал

### Завдання
- ✅ Створення завдання з назвою, описом, дедлайном
- 📬 Автоматичне сповіщення виконавця при призначенні
- 🔄 Зміна статусу: Очікує → В роботі → Виконано / Скасовано
- 💬 Коментарі до завдань (з сповіщенням сторін)
- 🗑 Видалення завдань (з підтвердженням)
- 📊 Статистика та фільтрація за статусом

### Групи
- 👥 Керівник групи формує свою групу з рядових
- ➕ Додавання / ➖ видалення учасників
- 📁 Адмін бачить всі групи

### Реєстрація
- /start — реєстрація нових користувачів
- Адмін призначає ролі через інтерфейс бота
- Сповіщення при отриманні ролі

## 🔧 Команди

| Команда | Опис |
|---------|------|
| `/start` | Реєстрація / головне меню |
| `/me` | Переглянути свій профіль |

## 📦 Залежності

- **aiogram 3.7** — асинхронний фреймворк для Telegram Bot API
- **motor** — асинхронний драйвер MongoDB
- **python-dotenv** — завантаження .env файлу
